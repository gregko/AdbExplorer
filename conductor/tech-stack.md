# AdbExplorer Technology Stack

## Core Technologies
- **C#:** The primary programming language used for the application logic.
- **.NET 8.0:** The target framework for the application, ensuring a modern and performant runtime environment.
- **WPF (Windows Presentation Foundation):** The UI framework used for building the Windows desktop application.

## Libraries & Frameworks
- **CommunityToolkit.Mvvm:** A modern, fast, and modular MVVM library used for implementing the Model-View-ViewModel architectural pattern.
- **System.Drawing.Common:** Used for handling icons and image-related operations.

## External Tools
- **ADB (Android Debug Bridge):** The core external dependency used for communicating with Android devices and emulators.

## Architectural Patterns
- **MVVM (Model-View-ViewModel):** The primary architecture used to separate the UI, logic, and data.
